# hugerain recreation

Фронтенд-реконструкция актуальной live-версии [hugerain.net](https://hugerain.net/).
Проект собран как простая статическая страница на HTML, CSS и JavaScript.

## Что реализовано

- hero-блок с видеофоном и плавающим меню
- секция с процессом работы над продуктом
- блок актуальных проектов с большими отступами и медиа
- промежуточная секция с декоративными карточками
- секция услуг с иллюстрациями
- сетка партнёров с логотипами и wordmark-карточками
- форма заявки и footer с дополнительной Telegram-формой
- scroll reveal-анимации, stagger-эффекты и hover-состояния почти на всех элементах
- адаптив под desktop, tablet и mobile

## Как запустить

Можно просто открыть `index.html` в браузере.

Если нужен локальный сервер:

```bash
python -m http.server 4173
```

После этого открыть `http://127.0.0.1:4173/`.

## Заметки

- Проект сверялся с текущей live-версией сайта, а не со старым кэшем или ранними скриншотами.
- Часть логотипов и изображений подгружается с публичного Framer CDN, чтобы визуально остаться ближе к оригиналу.
- Некоторые партнёрские wordmark-блоки воссозданы CSS-типографикой, если оригинальный логотип не был доступен как отдельный ассет.
